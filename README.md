# PRO-TIMER: Precision Edition

製造現場や生産ラインの進捗管理を効率化する、Webベースの多機能タイマー・計算ツールです。
生産条件を入力するだけで、終了予定時刻や箱ごとの完了タイミングを瞬時に計算し、可視化します。

**[📱 アプリを使用する (Live Demo)](https://dma-cmyk.github.io/mfg-timer-web2/)**

## ✨ 主な機能

### 1. 🧮 生産計算 (Calculator)
必要なパラメータを入力すると、リアルタイムで完了予定時刻を算出します。
- **入力項目**: 生産予定数、取り数、1箱の入数、サイクルタイム(秒)、稼働率(%)
- **出力**: 終了予定日時、総ショット数、総箱数、1箱あたりの所要時間
- **開始時刻調整**: 「今」ボタンで現在時刻をセット、または任意の日時を指定可能

### 2. ⏱️ 完了表 (Timeline)
箱ごとの完了予定時刻をリスト表示し、進捗管理をサポートします。
- **進捗記録**: 「今」ボタンを押すと、その箱の完了を現時刻として記録し、以降のスケジュールを自動再計算（シフト機能）。
- **微調整**: `+` `-` ボタンや時刻入力で、分単位の遅れや進みを反映。
- **メモ機能**: トラブルや休憩などの記録用に、箱ごとにメモを残せます。

### 3. ☁️ 履歴 & クラウド共有 (History & Cloud)
- **ローカル保存**: 計算結果をブラウザ内に保存し、後から確認・再開が可能。保存後の条件変更（再計算）にも対応しています。
- **クラウド共有**: グループIDを発行して、チームメンバーとリアルタイムで進捗を共有（Firebase利用）。QRコードでの簡単参加もサポート。
- **CSV連携**: データのCSVエクスポート・インポートに対応。
- **アラーム通知 & ログ**: 箱の完了が近づくと音声（TTS）でお知らせ。聞き逃しても「通知ログ」から履歴を確認できます。

### 4. 🎨 テーマ切り替え
作業環境や好みに合わせて、デザインを変更できます。
- 通常モード
- Cyberpunk 2077
- 丸の内OL
- 南国ビーチ
- ギャラクシー

### 5. ℹ️ その他
- **ヘルプ機能**: アプリ内のメニューから、詳しい操作ガイドをいつでも参照できます。

## 🛠️ 使用技術

- HTML5 / CSS3 (Tailwind CSS via CDN)
- JavaScript (Vanilla ES6+)
- Firebase (Authentication, Firestore)
- Google Fonts / Icons

## 📖 使い方

1. **アクセス**: [https://dma-cmyk.github.io/mfg-timer-web2/](https://dma-cmyk.github.io/mfg-timer-web2/) をブラウザで開きます。
2. **計算**: 「計算」タブで生産条件を入力します。
3. **確認**: 「完了表」タブでスケジュールの詳細を確認します。
4. **保存**: 右上の「保存」ボタンで履歴に残します。
5. **共有 (任意)**: 右上のメニューから「クラウド共有」を選び、グループを作成または参加することで、他の端末とデータを同期できます。
6. **ヘルプ**: 操作に迷ったらメニューの「ヘルプ」からガイドを確認できます。
